# PlayCS Tournament Rulebook

Official rules and regulations for Counter-Strike 2 tournaments operated by [PlayCS](https://playcs.gg), in association with the Israeli Gaming Association.

## Documents

- [**PlayCS-Tournament-Rulebook.md**](PlayCS-Tournament-Rulebook.md) — Full tournament rulebook (current version)

## About

PlayCS tournaments are classified as Tier 2 Ranked Tournaments under [Valve's Tournament Operating Requirements](https://github.com/ValveSoftware/counter-strike_rules_and_regs/blob/main/tournament-operation-requirements.md). This repository serves as the immutable, version-controlled publication of our rulebook in compliance with Valve's data verifiability requirements (Section 4.4).

All changes to the rulebook are tracked via Git history.
